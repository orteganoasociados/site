# Ortegano & Asociados - Escritorio Jurídico-Contable

Este repositorio contiene el sitio web oficial de **Ortegano & Asociados**, un escritorio profesional de vanguardia dedicado a brindar soluciones integrales en las áreas de contaduría pública, fiscalidad y asesoría legal mercantil en Venezuela.

## ⚖️ Nuestra Identidad

Ortegano & Asociados se especializa en el respaldo y blindaje de operaciones empresariales mediante el mayor rigor técnico y pulcritud ética. Nuestra firma une la precisión financiera con la seguridad jurídica para garantizar el éxito y la transparencia de nuestros clientes.

## 🛠️ Áreas de Especialización

### 📊 Contaduría Pública y Fiscalidad
- Contabilidad General y Fiscal (IVA, ISLR, Retenciones).
- Auditoría Financiera y de Gestión bajo normas VEN-NIF.
- Preparación y Dictamen de Estados Financieros.
- Auditoría preventiva de cumplimiento de deberes formales.

### 📜 Escritorio Jurídico
- Defensa y Asesoría Legal-Tributaria.
- Derecho Laboral y Corporativo.
- Redacción y visado de contratos mercantiles.
- Constitución y liquidación de sociedades.

## 💻 Detalles del Proyecto Web

El sitio está desarrollado con tecnologías modernas para garantizar un alto rendimiento y una experiencia de usuario premium.

### Tecnologías Utilizadas
- **React 19**
- **Vite**
- **TypeScript**
- **TailwindCSS**
- **Motion (Framer Motion)**
- **Lucide React**

### Instalación y Desarrollo

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/orteganoasociados/orteganoasociados_githubpages.git
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   ```
